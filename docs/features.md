# Feature Glossary & Leakage Review (FEAT-5)

This document provides a glossary of all features generated by the pipeline and documents their data sources, mathematical definitions, and pre-kickoff availability justifications to prevent lookahead data leakage.

---

## 🚦 Leakage Prevention Framework

To guarantee that the backtesting and paper-trading frameworks are free of lookahead bias, all features comply with the **Strict Chronological Shift Constraint**:
1. All matches are processed in order of their played date and time.
2. The rolling history for a team or pair at Match $M$ on Date $D$ is computed strictly from matches $0$ through $M-1$ played strictly before Date $D$ (`shift(1)` logic).
3. The outcome of the current match (goals scored, goals conceded, full-time result, points) is **never** included in the calculation of the features for that same match.

## 📂 Feature Glossary

### 1. Overall Rolling Form (FEAT-1)
These features capture a team's general momentum across their last $N$ matches (where $N = 5$ or $N = 10$), regardless of home/away venue.

| Feature Name | Type | Description | Source | Pre-Kickoff Availability Justification |
| :--- | :--- | :--- | :--- | :--- |
| `home_rolling_gs_N` | Float | Mean goals scored by the home team in its last $N$ matches. | `FTHG`/`FTAG` | Uses prior matches only. Shifted by 1. |
| `home_rolling_gc_N` | Float | Mean goals conceded by the home team in its last $N$ matches. | `FTHG`/`FTAG` | Uses prior matches only. Shifted by 1. |
| `home_rolling_pts_N`| Float | Mean points earned (3 for W, 1 for D, 0 for L) by the home team in its last $N$ matches. | `FTR` | Uses prior matches only. Shifted by 1. |
| `home_rolling_gd_N` | Float | Mean goal difference (scored - conceded) by the home team in its last $N$ matches. | `FTHG`/`FTAG` | Uses prior matches only. Shifted by 1. |
| `away_rolling_gs_N` | Float | Mean goals scored by the away team in its last $N$ matches. | `FTHG`/`FTAG` | Uses prior matches only. Shifted by 1. |
| `away_rolling_gc_N` | Float | Mean goals conceded by the away team in its last $N$ matches. | `FTHG`/`FTAG` | Uses prior matches only. Shifted by 1. |
| `away_rolling_pts_N`| Float | Mean points earned by the away team in its last $N$ matches. | `FTR` | Uses prior matches only. Shifted by 1. |
| `away_rolling_gd_N` | Float | Mean goal difference by the away team in its last $N$ matches. | `FTHG`/`FTAG` | Uses prior matches only. Shifted by 1. |

*Imputation Rule for Cold Starts (e.g. Promoted/Relegated Teams)*:
If a team has played fewer than $N$ matches (M < N), we blend their actual history with the prior-season division average:
$$\text{Feature} = \frac{\sum_{i=1}^{M} \text{Value}_i + (N - M) \times \text{Prior Average}}{N}$$
- Prior Goals Scored/Conceded: `1.35`
- Prior Points: `1.365`
- Prior Goal Difference: `0.0`

### 2. Venue-Specific Rolling Splits (FEAT-2)
These features capture a team's venue-specific strength (home-field advantage or away-game resilience) over their last $N$ matches played at the same venue type.

| Feature Name | Type | Description | Source | Pre-Kickoff Availability Justification |
| :--- | :--- | :--- | :--- | :--- |
| `home_venue_rolling_gs_N` | Float | Mean goals scored by the home team in its last $N$ *home* matches. | `FTHG` | Uses prior home matches only. Shifted by 1. |
| `home_venue_rolling_gc_N` | Float | Mean goals conceded by the home team in its last $N$ *home* matches. | `FTAG` | Uses prior home matches only. Shifted by 1. |
| `home_venue_rolling_pts_N`| Float | Mean points earned by the home team in its last $N$ *home* matches. | `FTR` | Uses prior home matches only. Shifted by 1. |
| `home_venue_rolling_gd_N` | Float | Mean goal difference by the home team in its last $N$ *home* matches. | `FTHG`/`FTAG` | Uses prior home matches only. Shifted by 1. |
| `away_venue_rolling_gs_N` | Float | Mean goals scored by the away team in its last $N$ *away* matches. | `FTAG` | Uses prior away matches only. Shifted by 1. |
| `away_venue_rolling_gc_N` | Float | Mean goals conceded by the away team in its last $N$ *away* matches. | `FTHG` | Uses prior away matches only. Shifted by 1. |
| `away_venue_rolling_pts_N`| Float | Mean points earned by the away team in its last $N$ *away* matches. | `FTR` | Uses prior away matches only. Shifted by 1. |
| `away_venue_rolling_gd_N` | Float | Mean goal difference by the away team in its last $N$ *away* matches. | `FTHG`/`FTAG` | Uses prior away matches only. Shifted by 1. |

*Venue-Specific Priors for Cold Starts*:
- **Home Team Venue Priors**: GS = `1.50`, GC = `1.20`, PTS = `1.60`, GD = `0.30`
- **Away Team Venue Priors**: GS = `1.20`, GC = `1.50`, PTS = `1.13`, GD = `-0.30`

### 3. Rest Days & Schedule Congestion (FEAT-3)
These features measure team fatigue and congestion in scheduling.

| Feature Name | Type | Description | Source | Pre-Kickoff Availability Justification |
| :--- | :--- | :--- | :--- | :--- |
| `home_rest_days` | Integer | Days between the home team's current match and their previous match (capped at 28). Defaults to 28 for cold starts. | `Date` | Calculated from prior match dates. |
| `away_rest_days` | Integer | Days between the away team's current match and their previous match (capped at 28). Defaults to 28 for cold starts. | `Date` | Calculated from prior match dates. |
| `home_is_congested` | Integer | Binary flag: `1` if home team rest days $\le 4$ (indicating mid-week or cup fixture congestion), else `0`. | `Date` | Calculated from prior match dates. |
| `away_is_congested` | Integer | Binary flag: `1` if away team rest days $\le 4$, else `0`. | `Date` | Calculated from prior match dates. |

### 4. Head-to-Head (H2H) History (FEAT-4)
These features capture historically specific matchup dynamics between the pair of teams.

| Feature Name | Type | Description | Source | Pre-Kickoff Availability Justification |
| :--- | :--- | :--- | :--- | :--- |
| `h2h_matches_played` | Integer | Number of prior meetings between the home team and away team in our 5-year timeline. | `HomeTeam`/`AwayTeam` | Uses prior matches between the pair only. |
| `h2h_home_goals_avg` | Float | Average goals scored by the current home team against the current away team in prior meetings. | `FTHG`/`FTAG` | Uses prior matches between the pair only. |
| `h2h_away_goals_avg` | Float | Average goals scored by the current away team against the current home team in prior meetings. | `FTHG`/`FTAG` | Uses prior matches between the pair only. |
| `h2h_home_pts_avg`  | Float | Average points earned by the current home team against the current away team in prior meetings. | `FTR` | Uses prior matches between the pair only. |

*First Meeting Rule*:
If the two teams have never met before in the 5-year timeline (`h2h_matches_played == 0`), these cells are left empty/null (`""` or `NaN`), allowing downstream modeling to explicitly handle first meetings without arbitrary imputation.
